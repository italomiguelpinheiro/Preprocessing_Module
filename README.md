# Preprocessing_Module

Preprocessing steps:
  1. One hot encoding
  2. Lower Text
  3. Word Tokenize
  4. Annonymation
  5. Spell Cheker
  6. Lemming
  7. Stemming
  8. Remove Stop Words
 
 # Supported languages
  1. English
  2. Spanish
  3. Portuguese
  4. French
  5. Japanese
  6. Chinese
  7. Korean
